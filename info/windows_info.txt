
### Windows 11 Terminal
- Neovim Statusline plugin config
- using https://github.com/nvim-lualine/lualine.nvim
    - this statusline has dependencies on Nerd Fonts ( https://www.nerdfonts.com/ )
    - the dependecy is: 
        {
            'nvim-lualine/lualine.nvim',
            dependencies = { 'nvim-tree/nvim-web-devicons' }
        }
    - nvim-tree/nvim-web-devicons : install this the usual way with Lazy.nvim
    - however, make sure to actually have the nerd font of your choice installed on the system
    - once font is installed, additional configuration for Windows Terminal is required
        - open Terminal -> Settings -> open JSON file
        - profile -> defaults -> font:
            "font": 
            {
                "face": "Google Sans Code, JetBrainsMono Nerd Font Mono",
                "size": 13
            },
        - In this case, JetBrainsMono Nerd Font Mono was used after Google Sans Code, 
            because the latter does not have a Nerd Font variant
    - once JSON is edited, save it, and restart the terminal
